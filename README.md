> **SmartCity AI** is an AI-powered urban intelligence platform that predicts future city development using Machine Learning. The system forecasts **Traffic Congestion**, **Air Quality (AQI)**, and **Population Growth** for **20 major Indian cities** through an interactive dashboard with charts, maps, and AI-generated insights.

---

## 📌 Overview

SmartCity AI enables users to analyze future urban growth by selecting a city, prediction year, and Machine Learning model. The platform visualizes predictions using interactive charts, KPI cards, Google Maps, and AI-powered recommendations. It is designed with a modular architecture and supports both **frontend-only execution** and an **optional Python backend** for advanced predictions.

---

## 🚀 Features

* 🏙️ Predict future city development
* 🤖 Multiple Machine Learning models
* 📈 Interactive Charts (Chart.js)
* 🗺️ Google Maps Integration
* 🌍 20 Major Indian Cities
* 📅 Forecasts from 2000–2100
* 💡 AI Recommendations & Risk Assessment
---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES2022)
* **Backend:** Python
* **Machine Learning:** Custom Prediction Models & Scikit-learn
* **Charts:** Chart.js
* **Maps:** Google Maps Embed API

---

## 📂 Project Structure

```text
SmartCityAI/
├── frontend/
│   ├── css/
│   ├── js/
│   └── index.html
├── backend/
│   ├── server.py
│   ├── ml_models.py
│   └── requirements.txt
└── README.md
```

---

## ⚙️ Getting Started

```bash
git clone https://github.com/Sameer0605/smart-city.git
cd smart-city
```

### Run Frontend

Open:

```text
frontend/index.html
```

### Run Backend

```bash
cd backend
python server.py
```
